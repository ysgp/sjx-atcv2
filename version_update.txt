SJX Training System - Version Update Log

v2.3.1 (2026-03-08)
--------------------------------------------------------------------------------
added:
- 密碼登入系統
  - 新增密碼登入方式（與 OAuth 登入並存，以 Tab 切換）
  - 預設密碼 SJX12345
  - 首次登入強制更改密碼（密碼長度至少 6 字元）
  - 密碼顯示/隱藏切換按鈕 (Eye/EyeOff)
  - Callsign 輸入自動轉為大寫
- /api/auth/password-login API - 密碼登入驗證
  - 驗證 Callsign 及密碼
  - 首次登入自動設定預設密碼 hash
  - 調用 vAMSYS API 自動連結 Discord（若尚未連結）
  - 連結 Discord 後自動獲取頭像、角色資訊
  - 自動判定教官權限（依 Discord 角色）
  - 更新 last_login_at 時間戳
  - 返回 requirePasswordChange 標記供前端判斷
- /api/auth/change-password API - 更改密碼
  - 驗證 session 及密碼長度
  - 禁止使用預設密碼
  - 更新 password_hash 並設定 password_changed = true

changed:
- OAuth 登入流程調整
  - 有學員 Role 但無學員記錄時，重導向至 /login?error=notify_admin
  - 移除 OnboardingModal 自助綁定流程（改為聯繫管理員）
- 登入頁面改版
  - 新增「密碼登入」與「OAuth 登入」Tab 切換
  - 密碼登入表單：Callsign 輸入框、密碼輸入框（含顯示切換）
  - OAuth 登入保留 Discord OAuth 按鈕
  - 首次登入更改密碼彈窗
  - 版本號更新至 v2.3.1

removed:
- OnboardingModal 帳號連結彈窗（改由管理員處理）
- vAMSYS OAuth 登入按鈕（OAuth Tab 僅保留 Discord）

database:
- sjx_students 新增欄位：
  - password_hash TEXT（bcrypt 加密密碼）
  - password_changed BOOLEAN DEFAULT FALSE（是否已更改預設密碼）
  - last_login_at TIMESTAMP（最後登入時間）

v2.3.0 (2026-03-08)
--------------------------------------------------------------------------------
added:
- 學員登入系統
  - Discord OAuth 登入
  - vAMSYS OAuth 登入（開發中）
  - 自動連結 Discord 帳號與 vAMSYS 飛行員資料
- 頁面存取權限控制
  - 未登入用戶導向登入頁面
  - 教官後台僅限教官角色存取
- 用戶下拉選單顯示 Callsign 及 Discord 帳號資訊
- 教官後台按鈕僅對教官角色顯示
- 個人資料頁面
  - 主要識別（Callsign、姓名、員工編號）
  - 資訊/狀態（培訓梯次、Discord ID、帳號狀態）
  - 時間（資料建立、Discord 連結、本次登入）
  - 鏈接狀態（Discord、vAMSYS 連結狀態）
  - 手動連結學員帳號功能（搜尋 Callsign 連結）
- vAMSYS 同步功能增強
  - 同步時自動獲取飛行員 Discord ID
  - 導入時自動連結 Discord 帳號
  - 更新現有學員的 Discord ID（如尚未連結）
  - 飛行員卡片顯示 Discord 連結狀態標籤
- 帳號連結引導流程 (OnboardingModal)
  - 未連結學員帳號時自動顯示引導彈窗
  - 步驟一：驗證身份（輸入 Callsign 和 vAMSYS 顯示名稱）
  - 步驟二：引導使用者連結 Discord
    - 方法一：前往 auth.vamsys.io/user/social 連結
    - 方法二：透過 Discord 頻道 🎫 Pilot Access 申請
- /api/auth/verify-vamsys API - 驗證 vAMSYS 飛行員身份
- 登入頁面錯誤訊息顯示（非學員登入顯示錯誤提示）

changed:
- 主頁移除教官後台直接入口（改由 Navbar 進入）
- Session API 返回完整用戶資訊
- 登入頁面改為全螢幕置中設計（不顯示 sidebar/navbar）
- /api/auth/logout 支援 GET 請求（直接訪問即可登出）
- 學員登入檢查：未在學員庫找到對應記錄時重導向至登入頁面並顯示錯誤
- 小考系統自動使用登入 Callsign，跳過身份驗證步驟直接進入章節選擇
- 結訓考試自動帶入登入 Callsign（唯讀顯示），直接按「開始考試」即可
- 成績查詢自動帶入登入 Callsign 並查詢成績
- 成績查詢頁面移除搜尋框，僅能查看自己的成績（隱私保護）
- 練習題章節（全無標準答案）處理改進
  - 小考：顯示「已完成」而非 FAILED，分數顯示為 100
  - 結訓考試：同樣處理（但通常不會有此情況）
  - 成績查詢：練習題顯示藍色「—」符號及「練習」標籤
- 學員登入改進：有學員 Role 的用戶即使 vAMSYS 未連結 Discord 也可登入
  - 登入後會顯示 OnboardingModal 引導綁定流程
  - TRAINEE_ROLE_IDS: 1471073994873634848, 1471161941153153068

removed:
- 成績查詢頁面的 Callsign 搜尋功能（僅能查看自己成績）

database:
- sjx_students 新增欄位：discord_id, discord_username, discord_avatar, discord_linked_at
